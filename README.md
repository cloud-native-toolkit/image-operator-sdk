# Operator SDK image

Container image for building operators in a CI pipeline. The image contains the following tools:

- podman - v2.1.1
- operator-sdk - v1.2.0
- ansible-operator - v1.2.0
- golang - v1.15.2
- oc cli - v4.5.9
- opm cli - v1.15.3
- kustomize - v3.8.4
- "Developer Tools" group (make, gcc, etc)
- openssl
