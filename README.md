# Operator SDK image

Container image for building operators in a CI pipeline. The image contains the following tools:

- podman - v2.0.6
- operator-sdk - v1.0.1
- golang - v1.15.2
- oc cli - v4.5.9
- "Developer Tools" group (make, gcc, etc)
- opm cli - v0.14.3
- kustomize - v3.8.4
- ansible-operator - v1.0.1
